# vencordautoinstall

## powershell script to automatically install vencord from source on windows

why does this exist?\
vencord sometimes removes itself on discord update, and I don't want to keep having to do these steps manually.\
note that doing this will deny you any support from the vencord discord server, and you have to "know what you're doing". if you don't, stick with regular vencord.\
\
maybe you find this helpful :)
